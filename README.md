# TaskNexus Platform ⚡

A premium, high-performance Task Management System built with **Next.js 15+** and **ASP.NET Core 9.0**. Featuring a stunning **Neon Purple Cyber-Industrial** theme and comprehensive team orchestration tools.

## ✨ Features

- **Project Clusters**: Organize tasks into high-level project nodes.
- **Orbital Task Matrix**: Kanban-style drag-and-drop task management.
- **Personnel Hub**: Manage team roles (Admin, Manager, Operative) and search operatives.
- **Time Logging**: Precise effort tracking for task sequences.
- **Notifications**: Real-time alerts for overdue operations.
- **Role-Based Access**: Secure identity management and access control.

## 🛠️ Technology Stack

- **Frontend**: Next.js (App Router), Lucide Icons, Bootstrap 5 (Custom Themed).
- **Backend**: ASP.NET Core Web API, Entity Framework Core.
- **Database**: SQLite (Development) / SQL Server (Production ready).

## 🚀 Getting Started

### 1. Prerequisites
- [.NET SDK 9.0+](https://dotnet.microsoft.com/download)
- [Node.js 18+](https://nodejs.org/)

### 2. Backend Setup
```bash
cd TaskBackend
dotnet build
dotnet run
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🏗️ Deployment

The project is structured as a monorepo. 
- The **Backend** serves api at `http://localhost:5129`
- The **Frontend** runs on `http://localhost:3000`
